# ルール配置振り分け

41の ai-rules を lint / hooks / ai-rules(docs) に振り分ける。

## 振り分け基準

| 配置先 | 基準 | 特徴 |
|---|---|---|
| lint (ESLint) | パターンマッチで自動検出できる | 最強。人もAIも従う |
| hooks (PreToolUse/PostToolUse) | lint では検出できないが強制したい | コミット前ブロック等 |
| ai-rules (docs) | 判断が必要で自動化できない | 設計方針、パターン選択 |
| 削除 | Claude が既に知っている / プロジェクト固有すぎる | 過剰指示を減らす |
