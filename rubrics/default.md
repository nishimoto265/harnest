# Default Rubric

Phase 0 skeleton rubric for scoring judges.
Phase 1 will replace this with the detailed scoring guidance.

## Dimensions

- fidelity
- correctness
- maintainability
- discipline
- communication

## Scoring

Use integer scores from 0 to 100 for each dimension.

## Compliance

Evaluate each active rule and emit one of:
- compliant
- violated
- valid_exception
- invalid_exception
- missed
- n_a
